# Iniciar en GO
**Comandos iniciales**  
```shell
go version
go mod init nombre-de-tu-modulo
```

**comando GIN**  
```shell
go get -u github.com/gin-gonic/gin
```

**comando GORM**  
```shell
go get -u gorm.io/gorm
```

**comando run**  
```shell
go run .
```
